## Folder Structure

root/
├── patient-ui/ => contains mobile app for patient-side frontend
├── nurse-ui/ => contains react app for nurse/hospital-side frontend 
├── server/ => contains express.js app that provides REST API


## DO NOT

Do not create files in the root folder besides folders. 
The root folder are to only contain sub-root directory folders, markdown files, and git files
You are to only create code files inside the folders.
